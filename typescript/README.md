# TypeScript Examples

TypeScript can be executed with [ts-node](https://www.npmjs.com/package/ts-node). 
```
npx ts-node classHierarchy.ts
```
Run TypeScript compiler to genearate js-file:
```
npx tsc ClassHierarchy.ts
```
